# Colgen — справочник по использованию

Ты — эксперт по colgen (https://github.com/vmkteam/colgen).
Документация: https://vmkteam.pages.dev/colgen/

## Обзор

Colgen — генератор коллекционных методов для Go-структур: извлечение полей, индексирование, группировка, конвертация между слоями. Работает через AST-парсинг: **код должен компилироваться перед генерацией**.

## Формат аннотаций

```go
//go:generate colgen [flags]
//colgen:Struct1,Struct2              // базовые: тип коллекции + IDs() + Index()
//colgen:Struct1:Field,Group(Name)    // расширенные генераторы
```

Флаги: `-imports=pkg/db` (импорты), `-funcpkg=reviewer` (пакет для Map/MapP), `-list` (суффикс "List" вместо "s").

## Базовый режим

`//colgen:Review` генерирует:
- `type Reviews []Review`
- `func (ll Reviews) IDs() []int` + `Index() map[int]Review`

Требует поле `ID` в структуре.

## Расширенные генераторы

Указываются после `:` через запятую: `//colgen:News:StatusID,Group(TagIDs),MapP(db)`

### `<Field>` — извлечение значений поля

Возвращает `[]T`. Поведение зависит от типа поля:
- `T` (scalar) — `r[i] = ll[i].Field`
- `*T` (pointer) — nil-check + dereference
- `[]T` (slice) — flatten: `append(r, ll[i].Field...)`

```go
//colgen:News:StatusID   // StatusID int    → StatusIDs() []int
//colgen:News:TagIDs     // TagIDs   []int  → TagIDs() []int (flatten)
//colgen:News:Author     // Author   *Author → Authors() []Author
```

### `Unique<Field>` — уникальные значения

Как `<Field>`, но с `map`-дедупликацией. Поддерживает scalar, `[]T` (двойной цикл), `*T` (nil-check).

```go
//colgen:News:UniqueTagIDs    // []int → UniqueTagIDs() []int
//colgen:News:UniqueAuthorID  // *int  → UniqueAuthorIDs() []int
```

### `Index(Field)` — индекс по полю

```go
//colgen:Episode:Index(MovieID)  // → IndexByMovieID() map[int]Episode
```

### `Group(Field)` — группировка

Scalar — стандартная `map[T]List`. Slice — fan-out (элемент попадает в несколько групп).

```go
//colgen:Issue:Group(ReviewFileID)  // int   → GroupByReviewFileID() map[int]Issues
//colgen:News:Group(TagIDs)         // []int → GroupByTagIDs() map[int]NewsList
```

### `Count(Field)` — подсчёт по значению

```go
//colgen:News:Count(StatusID)  // → CountByStatusID(v int) int
```

### `Fill(Target,Junction)` — гидрация связанных сущностей

Первый аргумент — поле-приёмник, второй — junction/FK. Режим определяется по типу junction:
- `[]int` — many-to-many: итерация + `append`
- `*int` — one-to-one: nil-check + присваивание указателя

```go
//colgen:News:Fill(Tags,TagIDs)      // TagIDs []int → FillTags(related Tags) NewsList
//colgen:News:Fill(Author,AuthorID)  // AuthorID *int → FillAuthor(related Authors) NewsList
```

### `Cast(Interface)` — приведение к интерфейсу

Структура должна реализовывать интерфейс через pointer receiver.

```go
//colgen:Movie:Cast(Restrictable)  // → Restrictables() []Restrictable
```

### `MapP(pkg)` / `Map(pkg)` — конвертеры между слоями

| Директива | Сигнатура конвертера | Результат |
|-----------|---------------------|-----------|
| `MapP(db)` | `NewStruct(*pkg.Struct) *Struct` | `NewStructs([]pkg.Struct) Structs` (public) |
| `Map(db)` | `NewStruct(pkg.Struct) Struct` | `NewStructs([]pkg.Struct) Structs` (public) |
| `mapp(pkg)` | `newStruct(*pkg.Struct) *Struct` | `newStructs([]pkg.Struct) []Struct` (private) |
| `mapp(pkg.Type)` | другой тип | `newStructs([]pkg.Type) []Struct` |

## Inline-режим

```go
//colgen@NewCall(db)                          // конструктор в исходном файле
//colgen@newUserSummary(newsportal.User,full,json)
```

## Хелперы MapP/Map

`MapP`/`Map` директивы требуют generic-хелперы в пакете (или доступные через `-funcpkg`). Если в `collection.go` их нет — предложи добавить:

```go
// MapP converts slice of type T to slice of type M with given converter with pointers.
func MapP[T, M any](a []T, f func(*T) *M) []M {
    n := make([]M, len(a))
    for i := range a {
        n[i] = *f(&a[i])
    }
    return n
}

// Map converts slice of type T to slice of type M with given converter.
func Map[T, M any](a []T, f func(T) M) []M {
    n := make([]M, len(a))
    for i := range a {
        n[i] = f(a[i])
    }
    return n
}
```

## Маркеры и правила

- Сгенерированные файлы `*_colgen.go` содержат заголовок `// Code generated by colgen; DO NOT EDIT.`
- Безопасно редактировать: `collection.go` (аннотации), файлы с конвертерами (`NewXxx`)
